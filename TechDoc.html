Introduction to HTML

HTML is a marking language; it can be used to write HTML files containing variety of HTML elements.

For example, <p> is an element standing for 'paragraphs, which are often used to hold text contents.'

Let's try writing the classical 'Hello World!' in HTML!

            <p>Hello World!</p>
  • <p> is the opening paragraph tag; it starts a p element.
  • </p> is the closing paragraph tag; its ends up a p element.
  • Any content put in between opening and closing tags, i.e. Hello World! in our case, belongs to the element.
    • Header Levels

    h1 to h6 are six levels of headings in HTML.

    By default, the font size is decreasing from h1 to h6, we can make that customized though.

                <h1>This is a level-1 header.</h1>
            <h2>This is a level-2 header.</h2>
            <h3>This is a level-3 header.</h3>
            <h4>This is a level-4 header.</h4>
            <h5>This is a level-5 header.</h5>
            <h6>This is a level-6 header.</h6>

    The output of the code above is:

    This is a level-1 header.

    This is a level-2 header.

    This is a level-3 header.

    This is a level-4 header.

    This is a level-5 header.
    This is a level-6 header.
    Embed an image in HTML

    We can embed an image from external resource with an img element and assign the url to its src attribute.

                <img src='https://upload.wikimedia.org/wikipedia/commons/thumb/8/80/Wikipedia-logo-v2.svg/300px-Wikipedia-logo-v2.svg.png' alt='wiki-logo'>

  • You might note that there's no closing img tag. Because img is a self-closing element.
  • Also, there's an alt attribute above; often we need an alternative text as a description of the image.
    • Embed a (url) link in HTML

    Now let's try to add a url link in the webpage.

    We can do this with an anchor(a) element.

                <a href='https://en.wikipedia.org/wiki/Wikipedia' target='_blank'>Wikipedia</a>

    The output of the code above is:

    Wikipedia

    The text between the anchor tags (Wikipedia) will be displayed.

    And the target attribute with a value of _blank will make the page opened in a new browser tab.

    Create an input in HTML

    There are types of inputs in HTML. Some frequently used ones are:

    • Text field, for users to enter text information;
    • Radio button, for users to select one of multiple options.
    • Checkbox, for users to select more than one of multiple options.
    • A type attribute is required to indicate which kind of input is to be used.
    • Often we need a label element to provide a short description for input fields.
    • If an input is mandatory, use required to indicate.
    • We can also use a placeholder to give users a hint regarding what to enter.
                <label>Name</label>
            <input type='text' placeholder='Your name'>
